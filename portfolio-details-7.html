Guess the Number: A Java Game of Chance with JavaFX and BlueJ

Project information

  • Category: Game Development
  • Client: Open Source
  • Project date: 2022
  • Project URL: github.com/IVANLANGA/randome_number.git

The objective of this game is to guess a random number between 1 and 10, providing an entertaining and interactive experience. Leveraging the power of JavaFX and the simplicity of BlueJ, I designed a visually appealing user interface with intuitive controls. Players can input their guesses and receive immediate feedback on whether their guess is too high or too low. Through this project, I gained hands-on experience in event handling, graphical user interface design, and implementing game logic. The combination of JavaFX's rich features and BlueJ's user-friendly environment enabled me to develop a polished and enjoyable gaming experience. This Java game showcases my skills in Java programming, problem-solving, and user interface design, while providing players with an engaging and challenging gameplay.
